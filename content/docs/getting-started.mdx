Transform any API into a paid USDC service in just 2 minutes with x402 Solana SDK. Compatible with x402 protocol and available on NPM.

## Prerequisites

Before you begin, make sure you have:

- **Node.js** 16+ installed
- **NPM** or **pnpm** package manager
- A **Solana wallet address** for receiving USDC payments
- Basic knowledge of **JavaScript/TypeScript**
- **Optional**: USDC tokens for testing payments

## Installation

Install the x402 Solana SDK - **single package only**:

```bash
npm install x402-solana-sdk
```

Or with pnpm:

```bash
pnpm add x402-solana-sdk
```

**That's it!** No multiple packages, no complex dependencies. Everything you need in one package.

## Quick Setup

### 1. Choose Your Framework

x402 SDK supports multiple frameworks. Choose the one you're using:

#### Express.js

```typescript
import express from 'express';
import { createX402Middleware } from 'x402-solana-sdk';

const app = express();

const x402 = createX402Middleware({
  amount: 0.10, // $0.10 USDC
  payToAddress: process.env.PAY_TO_ADDRESS,
  facilitatorUrl: 'https://x402.org/facilitator',
  network: 'solana-devnet'
});

// Public endpoint (free)
app.get('/public', (req, res) => {
  res.json({ message: 'This is free!' });
});

// Premium endpoint
app.get('/premium', x402.requirePayment(0.10), (req, res) => {
  res.json({
    message: 'Premium content unlocked!',
    payment: req.x402,
    amount: '$0.10 USDC'
  });
});

app.listen(3000);
```

#### Fastify

```typescript
import Fastify from 'fastify';
import x402Plugin from 'x402-framework-sdk/fastify';

const fastify = Fastify();

await fastify.register(x402Plugin, {
  facilitatorUrl: 'http://localhost:3001',
  merchantAddress: 'YOUR_SOLANA_ADDRESS'
});

// Premium endpoint
fastify.get('/premium', {
  x402: { amount: 0.01 }
}, async () => {
  return { 
    message: 'Premium content unlocked!',
    data: 'Exclusive content here...'
  };
});

await fastify.listen({ port: 3000 });
```

#### Hono

```typescript
import { Hono } from 'hono';
import { x402 } from 'x402-framework-sdk/hono';

const app = new Hono();

// Premium endpoint
app.get('/premium',
  x402({ 
    amount: 0.01,
    merchantAddress: 'YOUR_SOLANA_ADDRESS',
    facilitatorUrl: 'http://localhost:3001'
  }),
  (c) => c.json({
    message: 'Premium content unlocked!',
    data: 'Exclusive content here...'
  })
);

export default app;
```

### 2. Set Up Environment Variables

Create a `.env` file in your project root:

```env
# x402 Solana SDK Configuration
PAY_TO_ADDRESS=YOUR_SOLANA_ADDRESS_HERE
X402_FACILITATOR_URL=https://x402.org/facilitator
X402_NETWORK=solana-devnet

# Optional
X402_TIMEOUT=30000
X402_MAX_PAYMENT_AMOUNT=1000000000
```

### 3. Get Your Solana Address

If you don't have a Solana address yet:

```bash
# Install Solana CLI
npm install -g @solana/cli

# Generate new wallet
solana-keygen new --outfile ~/.config/solana/id.json

# Get your address
solana address
```

Copy this address to your `.env` file as `PAY_TO_ADDRESS`.

## Client Usage

Create a client to make payments:

```typescript
import { createClient, createWallet } from 'x402-solana-sdk';

// Create or load wallet
const wallet = createWallet(); // or X402Wallet.fromSecretKey()

// Create client
const client = createClient({
  wallet,
  facilitatorUrl: 'https://x402.org/facilitator',
  network: 'solana-devnet'
});

// Make payment request
const response = await client.payAndFetch('http://localhost:3000/premium', {
  amount: 0.10 // $0.10 USDC
});

console.log(response); // Premium content
```

## Testing Your Setup

### 1. Start Your Server

```bash
node server.js
# or
npm start
```

### 2. Test Public Endpoint

```bash
curl http://localhost:3000/public
```

Expected response:
```json
{
  "message": "This is free!"
}
```

### 3. Test Premium Endpoint (Without Payment)

```bash
curl http://localhost:3000/premium
```

Expected response:
```json
{
  "success": false,
  "error": "Payment required",
  "code": "PAYMENT_REQUIRED"
}
```

### 4. Test Premium Endpoint (With Payment)

Use the client code above or our test utility:

```bash
npx x402-client pay http://localhost:3000/premium 0.01
```

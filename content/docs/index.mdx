## Quick Start

Get started with x402 SDK in under 2 minutes:

```bash
npm install x402-framework-sdk x402-client
```

```typescript
import { createX402Middleware } from 'x402-framework-sdk/express';

const x402 = createX402Middleware({
  facilitatorUrl: 'http://localhost:3001',
  merchantAddress: 'YOUR_SOLANA_ADDRESS'
});

app.get('/premium', x402.requirePayment(0.01), handler);
```

## Quick Start

Get started with x402 Solana SDK in under 2 minutes:

```bash
# âœ… Single package installation
npm install x402-solana-sdk
```

```typescript
import { createX402Middleware, createClient, createWallet } from 'x402-solana-sdk';

// Server setup (Express.js)
const x402 = createX402Middleware({
  amount: 0.10, // $0.10 USDC
  payToAddress: process.env.PAY_TO_ADDRESS,
  facilitatorUrl: 'https://x402.org/facilitator',
  network: 'solana-devnet'
});

app.get('/premium', x402.requirePayment(0.10), (req, res) => {
  res.json({ 
    message: 'Premium content unlocked!',
    payment: req.x402 // Payment details
  });
});

// Client usage
const wallet = createWallet();
const client = createClient({ wallet });

const response = await client.payAndFetch('/premium', { amount: 0.10 });
```
